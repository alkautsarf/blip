blip.vim

Surface Claude Code's last turn, walking pet, and option pickers — all in your Mac notch.

┌──── blip.vim · pick ────┐
│ Which path do you want?
│ 1 ship it now
│ 2 add tests first
│ 3 back out
└──────────────────────────┘

For terminal-native devs

  • Never leave tmux to answer Claude's AskUserQuestion
  • ⌃⌥⇧ Enter jumps to the originating tmux pane
  • Vim keys (j/k) cycle options
  • Multi-session stack when 2+ sessions ping concurrently
  • Animated walking pet — claude-alive ambient presence

Install

brew tap alkautsarf/tap
brew install blip.vim
blip start
blip install
